生财有迹（Wealth Tracker）是一款专注于个人资产管理的应用程序，其核心功能是记录和展现用户的所有资产状况，以便提供一个清晰的财务现状概览。

## 项目愿景## 项目愿景

在当今多元化的经济环境中，个人财富管理变得日益重要。财富往往分散于各类金融账户和服务中，例如银行存款与理财产品、移动支付平台（如微信支付、支付宝）、公积金、医保账户、货币基金（例如余额宝）、债券、各种股票及基金产品、房地产、贵金属、外部借款（尽量不做此配置🤫）以及其他投资等等（对于部分朋友，或许还有贷款、欠款等负债）。这些账户中的财富数额在不断变动，使得快速准确地了解个人资产状况成为一项挑战。

本项目旨在提供一个高效、直观的解决方案，以应对个人财富管理中的分散性和复杂性。通过生财有迹，用户可以轻松掌握自己的财务状况，而无需记录每一笔具体的收支。其目标是通过简化操作流程，帮助用户揭示个人财富的整体变化趋势，并通过友好的用户界面，使用户能够轻松更新和管理自己的财务信息。

## 核心特性

- **简洁易用的操作界面**：用户可以通过几个简单的步骤快速上手，无需复杂的财务知识。
- **强大的数据可视化功能**：通过图表和图形，直观展示财富变化，帮助用户更好地理解和分析自己的财务状况。
- **本地部署的灵活性**：用户可以选择在本地部署应用，确保数据的私密性和安全性。
- **开源的代码架构**：项目的源代码完全开放，欢迎社区的贡献和协作，以不断改进和完善工具的功能。

通过这些特性，希望建立一个用户友好、透明且可信赖的个人财富管理工具，帮助用户更好地理解和管理自己的财务状况。

## 如何使用？

### 使用 Docker

**使用 [docker compose](https://docs.docker.com/compose/)**：

```yaml
version: '3.8'

services:
  wealth-tracker:
    image: wealth-tracker
    container_name: wealth-tracker
    ports:
      - '8888:8888'
    volumes:
      - ./data:/app/data
    restart: unless-stopped

volumes:
  data:
```

这将在后台启动服务，并且效果与下面的 `docker run` 命令相同。使用 Docker Compose 可以更方便地管理多个容器，并且配置更易读和维护。

**或者 docker run**：

```bash
docker run --name wealth-tracker -d -p 8888:8888 -v "$(pwd)/data:/app/data"  wealth-tracker
```

### 使用 Node.js

```bash
# clone project
git clone https://github.com/nicejade/wealth-tracker.git

# install & run for client
cd client && pnpm i && pnpm start

# install & run for srever
cd server && pnpm i && pnpm start
```

本项目客户端采用 [Svelte](https://svelte.dev/) 框架，基于 [Vite](https://vitejs.dev/) 所构建，默认 `5173` 端口，只需打开网址—— [http://localhost:5173](http://localhost:5173) 即可访问。
